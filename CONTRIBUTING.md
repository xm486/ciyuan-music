# 贡献指南

感谢你关注次元音乐开源版。

## 如何贡献

你可以通过以下方式参与：

- 提交 Bug 报告
- 提交功能建议
- 改进文档
- 修复问题并提交 Pull Request

## 开发建议

1. Fork 本仓库
2. 创建功能分支
3. 保持代码风格一致
4. 提交前检查 PHP / JS 语法
5. 提交 Pull Request 并说明修改内容

## 提交 Issue 时请说明

- 浏览器和设备
- PHP / MySQL 版本
- 复现步骤
- 错误截图或日志

## 注意事项

请不要在 Issue 或 PR 中提交：

- 数据库密码
- SMTP 授权码
- GitHub OAuth Secret
- 真实用户数据
- 受版权保护的音乐文件
